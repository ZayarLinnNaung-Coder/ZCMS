# ZCMSFrontEnd

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 14.0.3.

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`.

## Functionality

1. User Authentication
2. Content Manager
3. Media Library
4. User Setting
